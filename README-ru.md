# MyCosmo — плагин радиоастрономии для Minecraft

**MyCosmo** — это плагин для серверов на **Paper 1.21.11**, который добавляет в игру охоту за межзвёздными сигналами, запись на дискеты, улучшение их качества и интеграцию с игровой экономикой. Стройте радиотелескопы, сканируйте небо в поисках загадочных передач, сохраняйте их на дискеты, повышайте качество и продавайте за виртуальные деньги.

---

## Возможности

- 📡 **Радиотелескопы** — многоуровневые структуры для сканирования неба.
- 💾 **Дискеты** — записывайте сигналы на дискеты, каждая содержит случайное сообщение, источник и уровень качества.
- 🔍 **Источники сигналов** — десятки уникальных источников (планеты, реликтовые объекты, передатчики) со своими лорами.
- ⚙️ **Система качества** — дискеты имеют уровни качества (от «Ужасное» до «Отличное»), влияющие на читаемость и ценность.
- 🛠️ **Консоль улучшения** — улучшайте качество дискет, выбирая разные режимы.
- 💰 **Кассир / банкомат** — продавайте дискеты за внутриигровые деньги (требуется Vault).
- 🏛️ **Обсерватория** — храните и управляйте идентификаторами телескопов.
- 🌍 **Локализация** — полностью переводима (включены русский и английский языки).

## Требования

- **Paper 1.21.11** (или совместимый форк)
- **Java 21+**
- **Vault** (опционально, для экономики)

## Установка

1. Скачайте последний `MyCosmo.jar` со страницы релизов.
2. Поместите его в папку `plugins/` вашего сервера.
3. Запустите сервер. Плагин создаст конфигурационные файлы.
4. (Опционально) Отредактируйте `config.yml`, файлы в `lang/ru.yml` / `lang/en.yml` или `signals.yml` для настройки сообщений и источников.
5. Перезагрузите или перезапустите сервер.

## Начало работы

### 1. Постройте радиотелескоп

Телескоп представляет собой вертикальную структуру из 5 блоков:

| Слой (Y) | Блок                         |
|----------|------------------------------|
| +4       | Красный факел               |
| +3       | Вощёный громоотвод          |
| +2       | Вощёные медные прутья       |
| +1       | Красный камень (блок)       |
| 0 (основа) | **Синяя глазурованная терракота** |

При правильной установке основание станет интерактивным.

### 2. Сканирование сигналов

Нажмите правой кнопкой мыши по **блоку красного камня** (слой +1), чтобы начать сканирование.  
Телескоп будет слушать случайное время (от 5 до 15 секунд).  
Если небо над телескопом чистое, а калибровка достаточная, будет обнаружен сигнал.

> 💡 **Частота**, **направление** и **поляризация** сигнала должны совпадать с настройками в интерфейсе телескопа, чтобы его можно было записать.

### 3. Откройте интерфейс телескопа

Нажмите правой кнопкой мыши по **синей глазурованной терракоте** (основание), чтобы открыть меню телескопа.

В меню вы можете:
- Изменить частоту, направление и поляризацию.
- Запустить / остановить сканирование.
- Записать сигнал на пустую дискету.
- Увидеть уникальный ID телескопа (и скопировать его на бумагу).

### 4. Получите пустую дискету

Создайте пустую дискету по рецепту (бесформенный):

```
  Медный слиток  +  Золотой слиток  +  2 Красной пыли
```

(Порядок не важен.)

### 5. Запишите сигнал

Держа пустую дискету в инвентаре, откройте интерфейс телескопа, убедитесь, что параметры сигнала совпадают, и нажмите кнопку **«Записать»**.  
Запись длится около 11 секунд. После завершения в слоте дискеты появится заполненная дискета.

### 6. Используйте дискету

Возьмите дискету в руку и нажмите правой кнопкой мыши по воздуху или блоку (кроме некоторых «заблокированных» блоков, например, блоков красного камня, бочек и т.п.), чтобы **прочитать** её сообщение.  
Сообщение будет искажено в зависимости от качества дискеты. В чате отобразятся источник сигнала и сам текст.

### 7. Улучшение дискет

Постройте **консоль улучшения**, чтобы повышать качество дискет.

#### Структура консоли

Вид сверху (3×3×3):

**Слой Y+2 (верх)**  
```
[Железный блок] [Железный блок] [Железный блок]
```
**Слой Y+1**  
```
[Железный блок] [Железный блок] [Железный блок]
```
**Слой Y (основа)**  
```
[Блок красного камня] [Алмазный блок] [Блок красного камня]
```
(Или повёрнуто на 90° — алмазный блок в центре.)

Нажмите правой кнопкой мыши по **алмазному блоку**, чтобы открыть интерфейс консоли.

#### Режимы улучшения

| Режим        | Требуемое качество | Результат     |
|--------------|--------------------|---------------|
| Декодер      | (Зашифровано)      | Ужасное       |
| Шумоподавление| Ужасное            | Плохое        |
| Эквалайзер   | Плохое             | Среднее       |
| Динамический диапазон | Среднее     | Хорошее       |
| Мастеринг    | Хорошее            | Отличное      |

Поместите дискету в слот консоли, выберите правильный режим и подождите окончания улучшения (15 секунд).  
Вы можете подписаться на уведомления (иконка колокольчика), чтобы получать оповещения о завершении улучшения.

### 8. Продажа дискет (кассир)

Постройте **кассира** для продажи дискет за деньги.

#### Структура кассира

Вертикально:

- Основание: **Бочка**
- Середина: **Железные прутья**
- Верх (3 блока): **Железные прутья** в ряд (по оси X или Z).

Нажмите правой кнопкой мыши по бочке, чтобы открыть её инвентарь. Положите туда любые дискеты и закройте инвентарь — начнётся транзакция.  
Через несколько секунд (4–10 секунд) структура исчезнет с частицами, и вы получите оплату, основанную на ценности дискет.

**Формула ценности:** базовая цена (зависит от качества) + бонус источника (если есть).  
- Ужасное → 1, Плохое → 2, Среднее → 5, Хорошее → 7, Отличное → 12 (плюс бонус источника).

Требуется **Vault**. Если Vault не установлен, кассир предупредит об этом.

### 9. Обсерватория (хранение ID телескопов)

Постройте **обсерваторию**, чтобы сохранять бумаги с ID телескопов.

#### Структура обсерватории

- Основание (центр): **Светло-синяя глазурованная терракота**
- Слева/справа (или спереди/сзади) на том же Y: **Блок красного камня**
- Над ними (Y+1): **Железный блок** — покрытие 3×3 (включая центр).

Нажмите правой кнопкой мыши по основанию, чтобы открыть интерфейс с тремя слотами.  
Поместите в эти слоты бумаги с ID телескопа (получить можно, нажав правой кнопкой по краснокаменному блоку телескопа с бумагой в руке) — они сохранятся.

## Конфигурация

### `config.yml`

```yaml
locale: ru   # или en
```

Задаёт язык по умолчанию.

### `lang/`

- `ru.yml` – русские переводы.
- `en.yml` – английские переводы.

Вы можете добавлять или изменять любые сообщения.

### `signals.yml`

Определяет источники сигналов. Каждый источник содержит:
- `id` – уникальный идентификатор.
- `name` – отображаемое имя.
- `value_bonus` – дополнительная цена при продаже.
- `messages` – список возможных сообщений.

Вы можете добавлять новые источники или редактировать существующие.

## Локализация

Плагин использует встроенную систему локализации. Чтобы сменить язык, измените `locale` в `config.yml`.  
Все сообщения для игроков хранятся в папке `lang/`. Вы можете свободно их редактировать.

## Сборка из исходников

1. Клонируйте репозиторий.
2. Соберите с помощью Maven:

```bash
mvn clean package
```

JAR-файл будет в папке `target/`.
